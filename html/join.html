약관동의

[중요 안내] 본 약관은 팀 프로젝트 및 JavaScript 학습을 위한 더미 약관입니다. 실제 쇼핑몰, 실제 회사, 실제 서비스와는 관련이 없으며 법적 효력이 없습니다. 본 문서는 회원가입 화면 구현, 약관 동의 체크박스 검증, 스크롤 박스 UI 연습을 위해 작성된 예시 문서입니다.

제1조 목적

본 약관은 Mini Style Shop 예제 사이트의 회원가입, 서비스 이용, 상품 조회, 장바구니, 주문 정보 입력 등 학습용 기능의 이용 조건을 정하는 것을 목적으로 합니다.

제2조 용어의 정의

  1. “사이트”란 팀 프로젝트를 위해 제작된 가상의 쇼핑몰 웹페이지를 말합니다.
  2. “회원”이란 사이트의 회원가입 양식에 정보를 입력하고 가입 절차를 완료한 사용자를 말합니다.
  3. “상품”이란 사이트에 표시되는 의류, 잡화, 생활용품 등의 예시 데이터를 말합니다.
  4. “서비스”란 상품 조회, 장바구니 담기, 주문서 작성, 리뷰 작성, 마이페이지 확인 등의 학습용 기능을 말합니다.

제3조 약관의 효력

본 약관은 회원가입 화면에 표시됨으로써 효력이 있는 것처럼 구성되지만, 실제 법적 효력은 없습니다. 사용자는 본 약관이 프로젝트 과제용 더미 문서임을 이해한 후 동의 여부를 선택합니다.

제4조 회원가입

사용자는 이름, 아이디, 비밀번호, 이메일, 연락처, 주소 등의 정보를 입력하여 회원가입을 진행할 수 있습니다. 필수 항목을 입력하지 않은 경우 회원가입이 제한될 수 있습니다. 입력한 정보는 실제 서버로 전송되지 않으며, 브라우저의 localStorage에 학습 목적으로 저장될 수 있습니다. 사용자는 실제 개인정보가 아닌 예시 정보를 입력하는 것을 권장합니다.

제5조 아이디 및 비밀번호 관리

회원은 본인이 입력한 아이디와 비밀번호를 사용하여 로그인할 수 있습니다. 본 프로젝트에서는 보안 처리가 적용되지 않으므로 실제 사용하는 비밀번호를 입력하지 않아야 합니다. 비밀번호 확인 기능은 입력값 일치 여부를 확인하기 위한 JavaScript 검증 연습 목적으로 사용됩니다.

제6조 소셜 로그인 예시

카카오, 네이버, 구글 로그인 버튼은 실제 소셜 로그인 연동이 아닌 더미 기능입니다. 각 버튼은 예제 아이디와 예제 비밀번호를 입력하거나, 예제 회원 정보로 로그인된 것처럼 처리될 수 있습니다. 소셜 로그인으로 생성된 회원 정보 중 플랫폼에서 제공되지 않는 정보는 기본 예시 데이터로 대체됩니다.

제7조 주소 정보

회원가입 시 입력한 주소는 마이페이지에서 기본 주소로 표시될 수 있습니다. 상세주소는 선택 항목이며, 입력하지 않아도 회원가입을 진행할 수 있습니다. 본 사이트는 실제 배송을 수행하지 않으며, 주소 정보는 화면 출력 및 데이터 저장 연습을 위한 예시 정보입니다.

제8조 서비스 이용

회원은 사이트 내 상품 목록을 확인하고, 장바구니 또는 주문 관련 기능을 사용할 수 있습니다. 사이트에 표시되는 상품명, 가격, 이미지, 재고 등은 모두 예시 데이터입니다. 실제 결제, 배송, 환불, 교환은 이루어지지 않습니다.

제9조 회원 정보의 표시

회원가입 또는 로그인 후 마이페이지에서 이름, 아이디, 이메일, 연락처, 주소, 가입 방식 등의 정보가 표시될 수 있습니다. 표시되는 정보는 사용자가 입력한 값 또는 더미 로그인 기능을 통해 생성된 예시 값입니다. 본 프로젝트는 실제 개인정보 관리 기능을 제공하지 않습니다.

제10조 개인정보 처리

본 사이트는 실제 개인정보를 수집하거나 외부 서버에 저장하지 않습니다. 입력한 정보는 브라우저 저장소에 남을 수 있으므로, 공용 컴퓨터에서는 사용 후 데이터를 삭제하는 것을 권장합니다. localStorage에 저장된 정보는 브라우저 개발자 도구 또는 사이트 기능을 통해 삭제될 수 있습니다.

제11조 이용 제한

다음 각 호에 해당하는 경우 사이트 기능 이용이 제한될 수 있습니다.

  • 필수 입력값을 누락한 경우
  • 비밀번호와 비밀번호 확인 값이 일치하지 않는 경우
  • 약관 동의 체크박스를 선택하지 않은 경우
  • 올바르지 않은 형식의 이메일 또는 연락처를 입력한 경우
  • 프로젝트 목적과 맞지 않는 비정상적인 입력값을 사용한 경우

제12조 책임의 제한

본 사이트는 학습용 프로젝트이므로 실제 쇼핑몰 서비스 수준의 보안, 결제, 배송, 고객 지원 기능을 제공하지 않습니다. 사용자는 본 사이트의 모든 기능이 예시 구현임을 이해하고 이용해야 합니다.

제13조 약관의 변경

본 약관은 팀 프로젝트 진행 상황에 따라 임의로 수정될 수 있습니다. 수정된 약관은 화면에 표시되는 즉시 적용되는 것으로 간주합니다. 단, 본 약관은 실제 법적 효력을 가지지 않습니다.

제14조 기타

본 약관에 명시되지 않은 사항은 팀 프로젝트의 구현 목적과 수업 진행 기준에 따릅니다. 본 문서는 실제 약관이 아니며, 화면 구성과 JavaScript 검증 학습을 위한 예시 자료입니다.

부칙

본 더미 약관은 프로젝트 제작일로부터 적용되는 것으로 표시되며, 실제 서비스 약관으로 사용할 수 없습니다.
