팀원 A: 상품 출력 · 검색 · 필터 작업 방향

담당 페이지

  • index.html — 메인 / 추천 상품
  • products.html — 전체 상품 목록 / 검색 / 필터
  • product-detail.html — 상품 상세

핵심 방향

상품을 HTML에 직접 여러 개 작성하지 않고, 상품 데이터 배열을 만든 뒤 JavaScript로 화면에 출력하는 방식으로 진행한다.

상품 데이터 예시

const products = [
  {
    id: 1,
    name: "오버핏 후드티",
    category: "상의",
    price: 39000,
    color: "black",
    image: "hoodie.jpg",
    description: "편하게 입기 좋은 오버핏 후드티"
  }
];

products.html 작업

  • products 배열을 forEach로 반복 출력한다.
  • HTML에는 상품이 들어갈 영역만 만든다. 예: <div id="productList"></div>
  • 검색어를 입력하면 상품명에 검색어가 포함된 상품만 다시 출력한다.
  • 카테고리 버튼 또는 select로 상의 / 하의 / 가방 / 신발 / 전체 필터를 만든다.

검색 기능 방향

  • 입력한 검색어를 가져온다.
  • product.name.includes(searchText) 방식으로 상품명을 검사한다.
  • 검색 결과 배열만 다시 화면에 출력한다.

필터 기능 방향

  • 사용자가 선택한 카테고리 값을 가져온다.
  • product.category === selectedCategory 방식으로 비교한다.
  • 전체 선택 시 모든 상품을 다시 출력한다.

product-detail.html 작업

  • 상품 카드 클릭 시 product-detail.html?id=1 형태로 이동한다.
  • 상세 페이지에서 URLSearchParams로 id 값을 읽는다.
  • 상품 배열에서 id가 같은 상품을 찾아 상세 정보를 출력한다.
  • 이미지, 상품명, 가격, 카테고리, 설명, 수량 선택, 총 가격 표시를 넣으면 좋다.

index.html 작업

  • 추천 상품 3개 정도를 출력한다.
  • 랜덤 추천 또는 신상품 영역을 만들면 JS 사용이 분명해진다.
최소 완성 기준
  • 상품 배열 만들기
  • 상품 카드 반복 출력
  • 상품명 검색
  • 카테고리 필터
  • 상품 상세 페이지 출력
  • 메인 추천 상품 출력
전달 요약

A파트는 상품 데이터를 배열로 만들고, 그 배열을 기준으로 상품 목록, 검색, 필터, 상세 페이지를 구현한다. 검색은 상품명 includes, 필터는 category 비교, 상세 페이지는 id값으로 상품 하나를 찾아 출력하는 방식으로 진행한다.
